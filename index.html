Axelrod

This repository reproduces Axelrod's iterated prisoner's dilemma tournament. Contributions are welcome via pull request.

View the repository on Github

History of Axelrod's tournament

In the 1980s, professor of Political Science Robert Axelrod ran a tournament inviting strategies from collaborators all over the world for the Iterated Prisoner's Dilemma. You can read about this more here.

Contributing

The repository contains Python (2.7) code that reproduces the tournament. You can see the results from the latest run of the tournament here. All contributions are welcome: with a particular emphasis on contributing further strategies.

To contribute a strategy you need to do 4 (optionally 6) things (all of which can actually be done using the github website)

  1. Fork this repository
  2. Add a <strategy>.py file to the `strategies` directory. (Take a look at the others in there: you need to write code for the strategy and one other simple thing)
  3. Update the ./axelrod/strategies/__init__.py file (you need to write the import statement and add the strategy to the relevant python list)
  4. (This one is optional: write some unit tests in the ./axelrod/tests/` directory)
  5. (This one is also optional: add your name to the contributors list in the readme)
  6. Send a pull request
